## 功能

- 🗺️ 以点聚合形式展示中国星巴克门店分布
- 📍 以雷达点形式展示你当前所在位置
- 🧭 通过百度地图或高德地图快速导航至目标星巴克门店
- 🔍 支持按门店名称或地址信息模糊搜索
- 📋 展示每家星巴克门店的详情，如地址、营业时间、门店类型、门店标签等
- 🌓 支持深色与浅色两种主题
